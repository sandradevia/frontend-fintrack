#SIKEDA-MBG

SIKEDA MBG merupakan aplikasi web Sistem Informasi Keuangan Dapur yang dibangun untuk mendukung pengelolaan administrasi keuangan dan logistik bahan pangan pada lingkungan MBG/SPPG.

Aplikasi ini menyediakan fitur pengelolaan master bahan, pencatatan stok, mutasi barang masuk dan keluar, monitoring anggaran, serta penyusunan laporan keuangan secara terintegrasi. Dengan sistem ini, proses pencatatan yang sebelumnya manual dapat dilakukan secara digital, akurat, dan efisien.

SIKEDA MBG dikembangkan menggunakan framework Laravel dengan dukungan database PostgreSQL dan antarmuka dashboard admin modern berbasis Vite.
